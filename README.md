# Automation Testing Portfolio

Welcome to my comprehensive **Automation Testing Portfolio**. This repository is an enterprise-grade showcase of modern software quality engineering, demonstrating a progressive journey from foundational UI automation to complex, full-stack testing platforms.

## 🚀 Portfolio Vision
This portfolio is structured as a **monorepo**, housing five distinct, independent projects. Each project is designed to showcase specific expertise in the automation landscape, ensuring that recruiters and engineers can deep-dive into any layer of the stack.

---

## 📂 Project Directory

### 01. [E-Commerce Test Automation](./project-1-ecommerce-testing/)
**Focus**: UI/E2E Excellence & Page Object Model.
- **Tech Stack**: Playwright, TypeScript, POM.
- **Key Highlight**: A complete, scalable framework for the SauceDemo platform with cross-browser support and rich artifact management.

### 02. [REST API Testing Suite](./project-2-api-testing/)
**Focus**: Backend & Polyglot Automation.
- **Tech Stack**: Playwright (Request API), Python (Requests), Postman.
- **Key Highlight**: Dual-language implementation (TS/Python) demonstrating CRUD operations, contract testing, and response validation.

### 03. [CI/CD & Infrastructure](./project-3-cicd-pipeline/)
**Focus**: DevOps for QA.
- **Tech Stack**: Docker, Jenkins, GitHub Actions.
- **Key Highlight**: Containerized test runners and automated pipelines defined as code (`Jenkinsfile` and GitHub Workflows).

### 04. [Advanced Framework](./project-4-advanced-framework/)
**Focus**: Enterprise Architecture & Design Patterns.
- **Tech Stack**: TypeScript, PostgreSQL, Winston, Jest.
- **Key Highlight**: Implementation of **Singleton**, **Factory**, and **Strategy** patterns, integrated with a database for dynamic test data.

### 05. [Full-Stack Platform integration](./project-5-full-stack-platform/)
**Focus**: The Capstone Platform.
- **Tech Stack**: LambdaTest, K6, Prometheus, Grafana, Kubernetes.
- **Key Highlight**: A unified orchestrator coordinating E2E, API, and Performance testing with cloud grid integration and monitoring dashboards.
